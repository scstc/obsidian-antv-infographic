# AntV Infographic 示例

使用 `infographic` 代码块来渲染信息图。

## 基础列表

```infographic
infographic list-row-simple-horizontal-arrow
data
  lists
    - label 步骤 1
      desc 开始
    - label 步骤 2
      desc 进行中
    - label 步骤 3
      desc 完成
```

## 树形结构

```infographic
infographic hierarchy-structure-mirror
data
  title 28所联试平台AI智能体项目实施技术方案
  desc 基于国产信创环境的五层架构设计
  items
    - label 接入层
      children
        - label PC浏览器
        - label 语音输入SDK
        - label 移动端App
    - label 交互层
      children
        - label WebSocket
        - label HTTP API
    - label 业务层
      children
        - label 工作流引擎
        - label AI调度器
    - label 能力层
      children
        - label 大模型
        - label 向量数据库
        - label 文件处理
    - label 数据层
      children
        - label PostgreSQL
        - label Redis

```

## 简单列表

```infographic
infographic list-row-simple-vertical
data
  title 项目计划
  items
    - label 第一阶段
      desc 需求分析
    - label 第二阶段
      desc 系统设计
    - label 第三阶段
      desc 开发实现
    - label 第四阶段
      desc 测试验收
```

## 可用模板

根据 AntV Infographic 文档，可用的模板包括：

- `list-row-simple-horizontal-arrow` - 水平箭头列表
- `list-row-simple-vertical` - 垂直列表
- `hierarchy-structure-mirror` - 层级结构（镜像）
- `hierarchy-tree` - 树形层级
- `timeline` - 时间线
- 更多模板请参考：[AntV Infographic 模板](https://infographic.antv.vision/en/examples)
